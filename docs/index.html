Welcome To NDIToolbox

Open Source Data Analysis for Nondestructive Evaluation

NDIToolbox is a Python application designed to help with signal and image processing in Nondestructive Testing. Besides the usual read / preview / plot data routine, NDIToolbox also allows you to write your own data analysis scripts without worrying about putting together a GUI and the latest version includes the Probability Of Detection (POD) Toolkit that simulates statistical analysis of NDI measurements.

To begin with learning your way around NDIToolbox, the Getting Started page walks you through the basics of using NDIToolbox.

Visit NDIToolbox.com for the most up-to-date documentation and downloads.
